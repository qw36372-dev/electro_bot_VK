# ⚡ Electro Bot VK — Калькулятор электромонтажных работ

Бот ВКонтакте для сбора заявок и предварительного расчёта стоимости электромонтажных работ.
Полный аналог Telegram-версии на платформе ВК.

---

## 🚀 Быстрый старт

```bash
pip install -r requirements.txt
cp .env.example .env
# Заполните .env своими значениями
python main.py
```

---

## ⚙️ Переменные окружения

| Переменная | Описание |
|---|---|
| `API_TOKEN` | Токен сообщества ВК (Управление → Работа с API) |
| `ADMIN_IDS` | VK user_id администраторов через запятую |
| `LEADS_CHAT_ID` | peer_id беседы для заявок |
| `DATABASE_URL` | SQLite: `sqlite+aiosqlite:///bot.db` |

### Как узнать LEADS_CHAT_ID беседы:
- Беседа №1 → `2000000001`
- Беседа №5 → `2000000005`
- Формула: `2000000000 + номер беседы`

Бот должен быть **добавлен в беседу** и иметь права на отправку сообщений.

---

## 📋 Настройка сообщества ВК

1. Перейдите: Управление сообществом → Работа с API
2. Создайте ключ доступа с правами: **Сообщения сообщества**
3. Включите **Long Poll API** (версия 5.131+)
4. В разделе **Типы событий** включите: `Входящие сообщения`, `События сообщений`
5. Включите возможность писать боту в разделе **Сообщения**

## 📁 Структура проекта

```
electro_bot_vk/
├── main.py                  # Точка входа, роутер событий
├── config.py                # Конфигурация
├── states.py                # FSM-состояния (vkbottle)
├── keyboards.py             # Callback-клавиатуры
├── session.py               # Хранилище данных сессии пользователя
├── handlers/
│   ├── user/
│   │   ├── calculator.py    # Флоу опроса
│   │   └── confirm.py       # Подтверждение и отправка заявки
│   └── admin/
│       ├── admin_menu.py    # Панель администратора
│       └── prices.py        # Цены и коэффициенты
├── services/
│   ├── pricing.py           # Расчёт стоимости
│   └── lead_sender.py       # Отправка заявки в беседу
├── database/
│   ├── models.py            # SQLAlchemy-модели
│   └── crud.py              # CRUD-операции
└── utils/
    ├── validators.py        # Валидация ввода
    └── formatters.py        # Форматирование сообщений
```

## 👤 Команды пользователя

| Команда / текст | Действие |
|---|---|
| `/start` или `Начать` | Приветствие и запуск |
| `/cancel` или `Отмена` | Отмена текущего опроса |
| `/admin` | Админ-панель (только для ADMIN_IDS) |

---

## 🔑 Отличия от Telegram-версии

| | Telegram | ВКонтакте |
|---|---|---|
| Фреймворк | aiogram 3.x | vkbottle 4.x |
| Кнопки | Inline keyboard | Callback keyboard |
| Канал для заявок | Приватный канал | Беседа сообщества |
| Форматирование | HTML теги | Plain text |
| Удаление сообщений | `message.delete()` | `messages.edit()` (редактирование) |
| ID пользователя | Telegram user_id | VK peer_id |
